CallTrackingMetrics WP plugin

== Description ==

Call Tracking with CallTrackingMetrics provides the most detailed information
possible about who is calling from which advertising channel.  

We provide a built-in easy to use WordPress dashboard widget showing you call
volumn by day.

Discover which marketing sources drive the most calls to your business!

== Installation ==

CallTrackingMetrics is easy to use just install the plugin and copy the
embed code into your plugin.  Once it's installed start up your marketing
campaigns and you'll be able to discover your most effective marketing
sources.

To enable the new dashboard widget - go to account settings and enable the
API.   Copy the API keys from your CallTrackingMetrics account into the
settings area within the WordPress plugin.

The plugin settings page can be found under "Settings" ->
"CallTrackingMetrics".

= Features =

* Easy to configure
* Track calls from a full range of marketing sources
* Know how many of your calls are repeat
* Discover which marketing sources provide the best ROI
